# Modo de uso
1. Descarga la versión que se adecua a tu sistema operativo / procesador
2. Descarga la base de datos club.db
3. Ejecuta en una terminal el archivo binario.
4. El API está atendiendo requests!

Deberás asignarle permisos de ejecución al archivo binario en Linux y MacOS

Si quieres probar tu archivo openapi.yaml colócalo dentro de la carpeta docs/ con ese nombre
